Инструкция по работе с системой

Поиск
Поиск в базе данных осуществляется по фамилии (обязательна для заполнения)
Доступна опция поиска по точному совпадению. По умолчанию поиск ведется по подстроке
Поля Имя, Отчество, Год Описи - заполняются для уточнения запроса

Результаты поиска
Выводятся все найденные по запросу студенты
Щелкнув по результату можно просмотреть найденное изображение (опись)
Если у студента указана папка личного дела, то можно посмотреть ее содержимое

Просмотр записей
На вкладке "Все записи" выводятся все текущие записи из базы данных
Доступны фильтры:
- по первой букве фамилии
- по году описи
Можно сделать выгрузку указанных данных с сохранением их в файл
Доступны форматы:
- txt
- csv

Добавление описи
Есть возможность загрузить в базу данных пакет с новыми данными
Нужно загрузить файл в формате csv
Доступна пробная загрузка (без фиксации изменений в БД) и финальная
После загрузки сохраняется протокол результата с указанием результата проверки каждой записи
Загрузочный файл должен иметь 9 именованных столбцов (при этом порядок не важен):

Фамилия;Имя;Отчество;Опись;Дело;Год;Файл_описи;Страница;Примечание

Очистка описи
Позволяет удалить данные из БД по конкретной описи
Опись, подлежащая удалению выбирается из списка
По завершении операции будет сохранен отчет: указывается количество удаленных записей студентов и произошедшие ошибки
