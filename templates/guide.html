Инструкция по работе с системой

Поиск
Поиск в базе данных осуществляется по фамилии (обязательна для заполнения)
Доступна опция поиска по точному совпадению. По умолчанию поиск ведется по подстроке
Поля Имя, Отчество, Год Описи - заполняются для уточнения запроса

Результаты поиска
Выводятся все найденные по запросу студенты в виде списка
Щелкнув по результату можно просмотреть основную информацию о студенте и скан описи
Если опись еще не загружена, то выводится фиктивная опись
Если у студента указана папка личного дела, то будет показана ссылка на нее
По ссылке личного дела выводится список доступных к просмотру сканов из папки

Просмотр записей
В разделе "Все записи" выводятся все текущие записи из базы данных
Организован постраничный просмотр для удобства просмотра больших списков
Доступны фильтры:
- по первой букве фамилии
- по году описи
Можно сделать выгрузку указанных данных с сохранением их в файл
Доступные форматы для файла:
- txt
- csv

Добавление описи
В этом разделе сайта доступна пакетная загрузка данных в БД
Нужно загрузить файл на специальной форме раздела в формате csv
Доступна пробная загрузка (без фиксации изменений в БД) и финальная
После загрузки формируется протокол с указанием результата проверки каждой записи
Загрузочный файл должен иметь 9 именованных столбцов (при этом порядок не важен):

Фамилия;Имя;Отчество;Опись;Дело;Год;Файл_описи;Страница;Примечание;Папка_дела

Очистка описи
Позволяет удалить данные из БД по конкретной описи
Опись, подлежащая удалению выбирается из подготовленного списка
По завершении операции формируется протокол: указывается количество удаленных записей студентов и произошедшие ошибки

Обновление описи
Раздел предназначен для случая, когда изначально в пакетной загрузке не был указан файл описи
Опись, подлежащая обновлению выбирается из подготовленного списка
Далее указывается имя файла, который должен быть предварительно сохранен в общей папке архивной коллекции файлов
По завершении операции формируется протокол: фиксируются произошедшие ошибки и факт завершения операции

Добавление сканов в дело
В этом разделе производится добавление в БД папки личного дела студента
Необходимо заполнить основную информацию о студенте и указать имя папки, которую предполагается добавить указанному студенту
Папка должна быть предварительно сохранена в общей папке архивной коллекции
По завершении операции формируется протокол: фиксируются произошедшие ошибки и факт завершения операции
